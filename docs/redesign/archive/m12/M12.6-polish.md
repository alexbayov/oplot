## 1. Camera FX

Phaser camera умеет shake/flash/zoom. Применяем:

| Событие | Эффект |
|---|---|
| Crit hit | shake 100ms intensity 0.02 + flash white 50ms |
| Phase change моба | zoom 1.05 × 300ms + shake |
| Boss reveal | zoom + slow-mo 500ms + музыка swap |
| Hero low HP (< 20%) | red vignette overlay + pulse |
| Stim used | quick flash + zoom-out |
| Smoke escape | fade-to-grey + музыка fade |

Реализация — добавить `CombatCameraFX` модуль, эвенты engine публикуются через event bus.

## 3. Dynamic Music (4 слоя)

Текущая музыка боя — 1 трек loop'ом. Замена на **dynamic layered**:

| Слой | Когда играет | Перекрытие |
|---|---|---|
| **calm** | < 2 акторов engaged, или после kill всех живых | base |
| **combat** | бой в активной фазе (>= 2 акторов engaged) | base |
| **lowhp** | hero HP < 30% | over combat (filter sweep) |
| **boss** | boss мобы участвуют | replace combat |

Переходы — cross-fade 1-2 sec. Реализация через Howler.js (уже в проекте) либо Phaser Sound + tween volume.

### Тех-стек

- 1 audio file per layer (looping)
- ChannelMixer следит за `state.combatPhase` и выставляет volumes
- На phase change → мини fade-out music на 200ms (создаёт «вакуум»)

### Стоимость assets

3-4 трека (calm/combat/boss + lowhp фильтр).  Artist + Composer. Если Composer не доступен — использовать licensed loops (Soundsnap, AudioJungle).

## 4. Visual Polish

### Mob hit reaction
При попадании по мобу — sprite краткая trеmor + красный flash overlay (50ms).

### Death animation
Сейчас sprite просто исчезает. Заменить на 200ms fade + slight fall.

### Status icons animation
Иконки статусов появляются с pop-in (scale 0 → 1 в 150ms), исчезают с fade.

### Action bar
При активации приёма → button glow 300ms.

### HP bar
Уменьшение HP — анимированное (300ms ease-out), не моментальное.

## 6. Открытые вопросы

1. **Disable shake в settings?** Yes — а11y, мобильная адаптация.
2. **Mute by default на мобиле?** Yes (если первый запуск + mobile), иначе игрок просто муцит и теряет music.
3. **Music лицензия?** Если использовать AudioJungle — Alex купит лицензии. Если creative commons — без проблем.
4. **Phase change SFX отличается per mob?** Идеально — да. Реалистично — общий + 3 уникальных (бoss-mobs).
5. **Camera shake интенсивность на разных устройствах?** Подбирается экспериментально. На мобиле — слабее (укачивает).
