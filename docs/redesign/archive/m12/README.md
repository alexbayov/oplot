# M12 — archive

Эти документы спроектировали тактическую реформу боя (статусы, способности оружия, фазы мобов, инициатива-таймлайн, позиции, окружение).

В рамках **M13 (PR-1, 2026-06-13)** бой выпилен и заменён авторесолвом с decision surface (`src/systems/sortieResolve.ts`). Тактический бой возвращается отдельным треком в M14+ как развитие этих идей.

Документы оставлены не для исторической ценности, а как стартовая точка для M14: когда соберём «живую базу» и крафт, и решим, какая форма боя резонирует с игрой, эти спеки будут отправной точкой, а не разговоры с нуля.

См. `docs/redesign/M13-PIVOT.md` для контекста почему и `docs/GDD.md` для текущего шейпа боя (авторесолв).
