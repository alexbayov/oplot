## TL;DR

Все предметы получают **обязательное поле `tier: 1..5`**. Тир — это **единая ось прогрессии**, привязывающая:

- **Зоны** (где предмет добывается)
- **Мобов** (что они роняют)
- **Рецепты** (что и из чего крафтится)
- **UI** (рамка иконки, цвет, римская цифра)

Тир — это **не цвет редкости** (как в Diablo «зелёное/синее/жёлтое»). Это **уровень технологичности и опасности**. T5 не значит «легендарное», T5 значит «требует фабричных частей, добывается в реакторе».

## 1. Определения тиров

| Tier | Тематика | Откуда добывается | Игровая роль |
|---|---|---|---|
| **T1** | Подручное, из лома | Лес, подвалы, опушки | Стартовая экипировка, первые 2 уровня |
| **T2** | Гражданское, охотничье, простое огнестрельное | Пригород, склады, посёлки | Базовая ходовая экипировка |
| **T3** | Армейское советское, проверенное | Город, метро, заброшки | Mid-game, основная боевая экипировка |
| **T4** | Армейское современное, специализированное | Фабрика, госпиталь, режимные объекты | Late-game, требует целевого фарма |
| **T5** | Экспериментальное, штучное, трофейное | Endgame зоны (M9), уникальные данжи | Endgame билды, апогей прогрессии |

**Принцип:** между соседними тирами **должна быть видимая ступень урона/защиты**, но также **новые свойства** (не только цифры). Например:

- T1 → T2: появляется огнестрел вообще
- T2 → T3: появляется burst-режим (АКМ vs ПМ)
- T3 → T4: появляются модификации с серьёзными бонусами (ПСО-1 даёт +20% точность)
- T4 → T5: появляются уникальные эффекты (СВД с уникальным модом → возможно one-shot хед)

## 2. Полная привязка к контенту

### 2.1 Зоны → тиры мобов

| Зона | Mob tier | Player level recommend |
|---|---|---|
| Лес | T1–T2 | 1–3 |
| Подвалы / опушка | T1 | 1–2 |
| Пригород | T2 | 3–5 |
| Склады | T2–T3 | 4–6 |
| Город | T3 | 5–7 |
| Метро | T3–T4 | 6–8 |
| Фабрика | T4 | 7–9 |
| Госпиталь | T3–T4 | 7–9 |
| Бункер (endgame) | T4–T5 | 9–10 |
| Реактор (endgame, опционально) | T5 | 10 |

### 2.2 Mob tier → доступный дроп

**Правило:** моб роняет **только предметы своего тира или ниже**. С небольшим окном «overshoot» для редкого дропа.

| Mob tier | Гарантированный диапазон дропа | Редкий дроп (overshoot) |
|---|---|---|
| T1 | T1 | T2 (5%) |
| T2 | T1–T2 | T3 (3%) |
| T3 | T2–T3 | T4 (3%) |
| T4 | T3–T4 | T5 (2%) |
| T5 (босс) | T4–T5 | — (уже cap) |

→ Игрок **может** получить АК-74 с обычного армейца T3 (overshoot), но шанс мал. Гарантированный путь — фармить капитанов T4.

### 2.3 Tier → рецепты

Каждый рецепт получает поле `tier: 1..5`. Открытие рецептов:

- **T1 рецепты** — доступны с начала игры (~10 рецептов)
- **T2 рецепты** — открываются через узел «Барахольщик-1» в дереве навыков ИЛИ при первом крафте T1-узлового верстака
- **T3 рецепты** — узел «Барахольщик-3» ИЛИ найденный чертёж в зоне
- **T4 рецепты** — **только чертежи**, не получаются через перки. Чертежи падают с боссов зачисток
- **T5 рецепты** — **только уникальные чертежи** из endgame зон

**Принцип:** игрок не получает T4-крафт «автоматически» — за чертежом надо идти. Это создаёт **цели для зачисток**.

|---|---|---|---|
| T1 | Серая | `#808080` | Белый |
| T2 | Зелёная | `#3a8c4f` | Белый |
| T3 | Синяя | `#3a5f8c` | Белый |
| T4 | Фиолетовая | `#7a3a8c` | Жёлтый |
| T5 | Оранжевая | `#c46a1d` | Жёлтый |

**Римская цифра** (I, II, III, IV, V) — мелким шрифтом в **правом нижнем углу** иконки. Никаких звёздочек, плюсиков, ёлочек — только тир.

**Совместимость со стилем M10:** рамки соответствуют painted-стилистике, мягкие края, лёгкая фактура (а не флэт-углы как в Diablo).

На карте мира (painted illustration из M10) рядом с пином зоны — **цветная плашка тира**:

- `🟢 T2` — пригород
- `🔵 T3` — город
- `🟣 T4` — фабрика

Игрок видит тир до входа → понимает «я туда дойду или умру».

## 4. Recipe Tier Logic (правила)

**Правило 1: материалы тира `N` нужны только для крафта тира `N` и выше.**

- T1 рецепт = только T1 материалы (дерево, лом, ткань)
- T3 рецепт = T1–T2 материалы + **минимум 1 T3 материал** (электроника, армейский лом)
- T4 рецепт = T2–T3 материалы + **минимум 1 T4 редкий материал** (микросхема, военный пластид)

**Правило 2: чем выше тир — тем больше шагов крафта.**

- T1: 1 рецепт = готовое изделие
- T2: 1 рецепт = готовое изделие, но 2-3 ингредиента
- T3: иногда **промежуточный продукт** («заготовка», «обработанная пластина»)
- T4: цепочка из 2 рецептов (заготовка → готовое изделие)

**Правило 3: время крафта T-зависимое.**

| Tier | Craft time (in-game, между ходами) |
|---|---|
| T1 | 0 (мгновенно) |
| T2 | 0 |
| T3 | 1 час игрового времени (если будет hunger/time) |
| T4 | 3 часа |
| T5 | 6 часов |

Если механики времени нет — `craft_time_s: 0` для всех, поле остаётся как placeholder для будущей фазы M13 «Time pressure».

## 5. Миграция существующих предметов

Все 80 существующих предметов получают тир через таблицу присвоения. Делается в **первом PR M11.0a** заодно с миграцией модели.

**Пример таблицы присвоения** (в `src/state/migrations.ts` или `content/_tier_assignment.json`):

```json
{
  "wood": 1, "scrap": 1, "cloth": 1, "leather": 1, "rope": 1,
  "gunpowder": 2, "electronics": 3, "circuits": 4,
  "shiv": 1, "pipe": 1, "spear": 1, "machete": 2,
  "sawed_off": 2, "pipe_rifle": 2, "crossbow": 2,
  "hunting_rifle": 3, "rocket_launcher": 4,
  "first_aid": 2, "bandage": 1, "stim": 3, "medkit": 3,
  "rag_armor": 1, "leather_jacket": 2, "kevlar_vest": 3, "armored_vest": 4
}
```

Принцип присвоения:
- Самоделы из лома → T1–T2
- Реальные охотничьи → T2–T3
- Армейская экипировка → T3–T4
- Уникальные / endgame → T4–T5

## 9. Открытые вопросы

1. **Цвета рамок** — нужен ли отдельный палитра-pass с художником, или дефолтные хексы из этого дока ОК?
2. **«Overshoot» дропа** — фиксировать 2–5% универсально или варьировать (например, у спец-мобов выше шанс)?
3. **T5 контент** — сколько T5 предметов реально нужно на старте M11? Может быть достаточно 2–3 эталонов (СВД-комплект, уникальная броня), а остальное доделаем после M11.3 (зачистки) когда станет понятно что класть в боссы?
4. **Tier на расходники** (бинт, аптечка, медкит) — тир = «эффективность» или просто этикетка? Если эффективность — нужна явная разница (T1 бинт лечит 10 HP, T3 медкит лечит 50 HP). Это уже балансировка.
5. **Сортировка инвентаря** — дефолт «по типу» или «по тиру»?
