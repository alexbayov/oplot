## TL;DR

Бой получает **третью пространственную ось**: 3 ряда (front / mid / back) + 0-3 интерактивных объекта окружения (бочка, машина, тело).

Это **самая рискованная подвеха M12** — крупный UX перепил. Делаем отдельным prototype-PR перед интеграцией.

**Длительность:** 3 недели, 3 PR.

---

## 1. Three Lines (3 ряда)

Поле боя разделено на 3 ряда по глубине:

```
[BACK]   снайпер   sniper2          ← мобы snipers, scopes
[MID]    marauder  mutant  pack_rat ← универсальный (большинство)
[FRONT]  dog       boss    ▲hero    ← мили, дробовики
```

### Правила weapons × position

| Weapon class | Front | Mid | Back |
|---|---|---|---|
| Melee | ×1.5 damage | ×0.5 damage | недоступно |
| Shotgun | ×1.5 damage | normal | ×0.5 damage |
| Pistol | normal | normal | normal |
| Carbine/SMG | ×0.8 | normal | ×0.8 |
| Sniper | недоступно | ×0.8 | ×1.5 damage |
| Throw (молотов) | normal | normal | normal (но дальность 2 ряда) |

### Move action

- **Move action** = 1 main action. Перемещение на 1 ряд за ход.
- Свободный move (perk «Lightfoot»): 1 ряд без траты action.

### Cover вместо button

Существующая **УКРЫТИЕ** button заменяется: укрытие зависит от **environment object** в твоём ряду. Если в твоём ряду машина — +30% defense за ней. Если ничего нет — кнопка disabled.

## 2. Environment Objects

Каждая боевая встреча генерируется с **0-3 объектами** окружения:

| ID | Объект | HP | Эффект пассивный | Эффект активный (action) |
|---|---|---|---|---|
| `env_barrel` | Бочка | 10 | — | взорвать → AOE 15 dmg всем в ряду |
| `env_car` | Машина | 50 | +30% defense пока за ней | — |
| `env_corpse` | Труп | — | — | обыскать → 1 случайный ресурс (1 action) |
| `env_crate` | Ящик | — | блокирует line of sight | разломать → +scrap (1 action) |
| `env_radio` | Радио | 5 | — | включить → -2 turn fear на всех мобов |
| `env_trap` | Капкан | — | если моб встаёт на ряд → stun | разрядить → +scrap |

### Распределение

| Зона | env_objects density |
|---|---|
| Forest | 0-1 (только trap, corpse) |
| Suburb | 1-2 (corpse, crate) |
| City | 2-3 (car, barrel, corpse) |
| Hospital | 2-3 (corpse, radio, crate) |
| Factory | 3 (barrel, crate, trap) |
| Metro | 2 (radio, trap) |
| Reactor | 2-3 (barrel, radio, trap) |

Content Designer добавляет в `content/encounters.json` поле `env_pool: string[]`.

## 3. UI Изменения

### 3.1 Combat layout

Сейчас: горизонтальная панель портретов внизу.
Будет: **сетка 3×N** где строки = positions, столбцы = акторы в этом ряду.

```
                  [моб] [моб]          ← back row
       [моб] [моб] [моб] [моб]          ← mid row
              [моб] [hero]              ← front row
```

### 3.2 Move action

Action bar получает кнопку «MOVE ▲▼» которая открывает submenu:
```
[← Назад]      ← в back ряд
[Здесь]
[Вперёд →]    ← в front ряд
```

### 3.3 Environment objects

Размещены как **decorative tiles** между рядами. Hover → tooltip. Click если interactable → action option.

## 4. State Integration

```ts
interface PositionedActor extends CombatActor {
  position: 'front' | 'mid' | 'back';
}

interface EnvObject {
  id: string;
  kind: 'barrel' | 'car' | 'corpse' | 'crate' | 'radio' | 'trap';
  position: 'front' | 'mid' | 'back';
  hp: number | null;     // null = not destructible
  state: 'intact' | 'used' | 'destroyed';
  effect?: EnvEffectDef;
}

interface CombatState {
  // ... existing
  environment: EnvObject[];
}
```

CombatEngine: новые actions `move`, `useEnv`, `attackEnv`.

## 6. Открытые вопросы

1. **Diagonal targeting?** Можно ли из back бить в front? Yes для дальнего оружия, no для мили (нужно сначала move).
2. **Mob initial position** случайный или фиксированный? Фиксированный per encounter (Content prescribes).
3. **Свободный move как perk одинаково для героя и мобов?** Нет, только герой получает «Lightfoot». Мобы — каждый по своему AI.
4. **3 ряда выглядят на мобиле?** Тестировать в M12.3a — это критическая точка отказа.
5. **Сколько env_objects максимум одновременно?** 3. Более — UI перегружен.
6. **Можно ли env_object добавиться в бою?** Нет, статично. (Если делать дроп оружия от убитого моба как env_object — это M13.)
