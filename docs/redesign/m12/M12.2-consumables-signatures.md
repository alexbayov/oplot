## TL;DR

Три расходника + у каждого моба свой **signature ability** + видимый telegraph над портретом. После M12.2 каждый бой воспринимается уникально (моб ведёт себя характерно), а игрок получает возможность реагировать на угрозы (telegraph).

**Длительность:** 2 недели, 2 PR.

---

## 1. Три расходника-усилителя

| ID | Название | Эффект | Стак |
|---|---|---|---|
| `combat_stim` | Боевой стим | +1 main action в этом ходу, +2 turn frenzy | 5 |
| `bandage_field` | Полевая перевязка | снимает bleed, +20 HP, тратит 1 main action | 10 |
| `smoke_grenade` | Дымовая шашка | escape без HP-потерь (мобы теряют ход) | 3 |

### Дополнительные (если успеваем):
- `bullet_super` — патрон-усилитель: следующий выстрел +50% damage (Soulshot)
- `adrenaline_shot` — мгновенный +30 HP, но -5 HP/ход на 3 хода (overdose)
- `flare_distress` — призывает 1 союзника-наёмника на 3 хода (если торговец-радист купил у тебя)

Расходники крафтятся (рецепты в M12 content drop):
- стим: 2 medicine + 1 microcircuit
- бинт: 3 cloth + 1 medicine
- дымовая: 2 powder + 1 cloth + 1 scrap

## 2. Mob Signature Abilities

Каждому из 11 мобов даём **одну подпись** — особенность, которая делает бой с ним характерным:

| Mob | Signature | Эффект |
|---|---|---|
| marauder | Стрельба пачкой | если в комнате 2+ мародёра — pack_bonus +20% damage |
| wild_dog | Прокусить | 30% шанс bleed при атаке |
| mutant | Рёв | раз за бой — fear на всех в front position |
| looter_sniper | Прицел | telegraph «🎯» за 1 ход до crit-выстрела (×3 damage) |
| armored_guard | Тяжёлая броня | -50% damage от калибров < 7.62 |
| fanatic_berserker | Безумие | при HP < 50% — +50% damage, не уворачивается |
| pack_rat | Стая | при смерти одного — призыв +1 |
| relic_drone | Хайтек | устойчив к фишическим, уязвим к ЭМИ (новый расходник «emp_charge») |
| forest_alpha_mutant | Альфа | каждые 3 хода — рёв (AOE fear + heal себе 20 HP) |
| warehouse_drone_prime | Прайм | shield phase: -90% damage пока shield не сломать (3 удара) |
| city_guard_captain | Лидер | если в бою с ним — +20% accuracy для других мобов |

Content Designer прописывает в `mobs.json`:

```json
{
  "id": "looter_sniper",
  "signature_ability": {
    "id": "sniper_telegraph",
    "trigger": "every_3_turns",
    "telegraph_turns": 1,
    "effect_on_resolve": { "type": "crit", "multiplier": 3.0 }
  }
}
```

## 3. Telegraph UI

Над портретом моба — **иконка телеграфа** показывающая что моб готовит:

| Иконка | Что значит |
|---|---|
| 🎯 | целится в тебя — следующий ход crit |
| 🛡️ | взял укрытие — defense +50% |
| 💨 | бежит — отступает / уходит из боя |
| 💢 | в ярости — frenzy активирован |
| 🔥 | готовит AOE — следующий ход атакует всех в position |
| 📞 | вызывает подмогу — следующий ход +1 моб |
| ✨ | заряжает skill — следующий ход signature ability |

Игрок видит telegraph → решает: укрыться, убить первым, выпить стим, отступить.

## 6. Открытые вопросы

1. **Telegraph можно «сбить»?** Если убить моба в telegraph-турне — ability отменяется (это уже механика). Если оглушить (stun) — telegraph переносится на 1 ход или отменяется?
2. **Сколько расходников на старте?** Стартовый набор: 2 бинта, 1 дымовая. Стим — крафтится позже.
3. **Pack_bonus marauder работает между ходами?** Да, постоянный пассив пока 2+ мародёра живы.
4. **Telegraph виден на мобиле?** Yes — иконка 24px над портретом, всегда видна.
5. **Можно ли telegraph fake?** Сложно. На M12 — нет. В M13 — возможно для боссов.
