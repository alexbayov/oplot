## TL;DR

**Крит-система:** базовый шанс из оружия + scope + telegraph бонусы. Crit = ×2.5 damage + автостатус.
**Лут per kill:** добыча с каждого моба сразу после его смерти, опционально подобрать в бою (mid-fight loot).

**Длительность:** 1.5 недели, 2 PR.

---

## 1. Крит-система

### Формула crit chance

```
crit_chance =
  weapon.baseCritChance      (5% для пистолетов, 8% для винтовок, 15% для снайперок)
  + mod.scope.critBonus      (+10-20% если установлен прицел)
  + position.bonus           (+10% если в укрытии, +20% после telegraph «целюсь»)
  + perk.critBonus           (+5% от «Меткий стрелок» в M11.4)
```

При срабатывании:
- `damage × 2.5`
- автостатус: пистолеты/винтовки → expose; дробовики → stun; снайперки → expose + fear

### По оружию

| Weapon | Base crit | Crit special |
|---|---|---|
| Knife / melee | 5% | bleed auto |
| Sawed-off | 3% | stun auto |
| Pistols | 5% | expose |
| SMG (PPSh) | 5% | — |
| Carbines (SKS) | 8% | expose |
| AKM/AK-74 | 8% | expose |
| Mosin | 15% | expose + fear |
| SVD | 18% | expose + fear |
| Shotguns | 6% | stun |

### UI крита

При попадании crit:
- Цифра damage **золотая** + 1.5× больше шрифт
- Sound «headshot»
- Camera shake (M12.6 hook)
- Текст «КРИТ!»

## 2. Лут per kill

### Сейчас
Лут раздаётся в `LootScene` после ВСЕГО боя. Игрок не знает что с какого моба.

### Будет
Каждый моб при смерти **роняет лут на земле** (как env_object типа `corpse`). Игрок может:
- Подобрать сразу (1 main action) — лут попадает в backpack
- Подобрать в конце боя (default, как сейчас) — авто-добавляется при victory
- Оставить (если weight high)

### UI
Над трупом моба — иконка лута («лежит ПМ»). Click → подобрать (если хватает веса).

### Тактика
Игрок решает: атаковать другого моба или потратить ход на подбор. Создаёт **trade-off** между скоростью и жадностью.

## 3. State Integration

```ts
interface MobActor extends CombatActor {
  loot: InventoryStack[];   // pre-rolled при создании
  lootPickedUp: boolean;
}

interface CombatState {
  // ...
  groundLoot: { mobId: string, items: InventoryStack[] }[];
}
```

При смерти моба → `state.groundLoot.push({ mobId, items: mob.loot })`. При victory → если есть `groundLoot` неподобранный, авто-добавляется в backpack.

## 5. Открытые вопросы

1. **Crit cap?** Да, max 50% (иначе с прокачкой и scope скучно).
2. **Crit на расходники (молотов)?** Нет, AOE damage без crit (иначе imba).
3. **Лут лежит после ухода из боя или забирается?** Если retreat — теряется (incentive воевать до конца).
4. **Видимо ли заранее что упадёт?** Нет, drop-table скрыт. Но игрок учит мобов («с marauder обычно amma + scrap»).
