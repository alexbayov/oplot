## 1. Диагноз

### Главная проблема — не код. Это арт-пайплайн.

Все спрайты и фоны сейчас генерятся через `tools/art/gen_m*_assets.py` (Pillow).
Это было верное решение для M1 — разблокировать Engineer'а заглушками. Но оно
**фундаментально не масштабируется в красоту**:

| Ассет | Что было | Уровень |
|---|---|---|
| Hero | силуэт из прямоугольников + ружьё палкой | детский Newgrounds |
| Мобы | круг + красный прямоугольник = «мародёр» | placeholder |
| Иконки предметов | геометрия с обводкой, арбалет = зелёный крест | нечитаемо |
| Фоны зон | треугольники-ёлочки | iOS 2010 |

Сколько ни переписывай `sceneUi.ts`, кружок-мародёр останется кружком-мародёром.

### Что в плюсе

- Код чистый, 213 тестов проходят, кодовая база модульная
- Style-guide логичный (палитра, шрифты, размеры) — его можно сохранить
- Pillow-ассеты воспроизводимо генерятся → легко выкинуть целиком
- Уже есть детальный `docs/redesign-master-plan.md` (1186 строк) на UI/UX

Это редкая стартовая позиция: технически готовая игра + чистая база для замены ассетов.

## 2. Что меняется

**Арт-пайплайн меняется с Pillow на AI-генерацию + ручной cleanup.**

Style-guide это уже предусматривает в секции «M2+». Сейчас просто фактически
переходим. ДНК стиля сохраняется (Military Graphic Novel × Darkest Dungeon),
но исполнение становится honest painted, а не геометрические placeholder.

| | Старый пайплайн (Pillow) | Новый пайплайн (Painted) |
|---|---|---|
| Источник | Python-генератор фигур | Gemini/Nano Banana с фикс-формулой промпта |
| Обработка | прямой PNG | rembg (вырезка фона) → trim → square → resize |
| Размеры | 128×128 (sprite), 800×600 (bg), 64×64 (icon) | те же — drop-in замена |
| Стиль | стилизованный лоу-фай | painted graphic novel, ink outlines, chiaroscuro |
| Качество | МVP-приемлемо | релизное |

Размеры и форматы остаются идентичными → **код сцен не трогается** для замены
визуала. UI/landscape-редизайн — отдельный этап.

## 3. Утверждённое направление: «Painted Graphic Novel»

Конкретно — это:

- Painterly illustration с thick black ink outlines (~2-3 px)
- Dramatic chiaroscuro lighting (одно направленное освещение, тени по форме)
- Палитра: dark olive, rust brown, charcoal, sandy beige, amber/blood-red accent
- Атмосфера: Darkest Dungeon × Metro 2033 × Atomic Heart
- Контраст: высокий, тёмные среды + светлые акценты на персонажах

Полные формулы промптов: `ARTS_BIBLE.md`.

### Утверждённые референсы v1
