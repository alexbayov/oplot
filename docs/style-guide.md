# Style Guide — «Оплот»

> Визуальные стандарты проекта. Финализирован Artist-сессией на M1.
>
> Источник правды для визуального стиля. Все ассеты в `assets/` обязаны соответствовать этому документу. Конфликты со старым контентом разрешаются в пользу этого файла.

## Визуальный стиль

**"Military Graphic Novel"**

- Атмосфера: Darkest Dungeon (контрастные тени, тяжёлая палитра, угрожающая мрачность) + современный военный/постапокалиптический сеттинг (This War of Mine, Day R, Tarkov).
- Техника: painted / illustrated, полуреалистичный рисунок с претензией на реал.
- Обводки: bold чёрные контуры 2–3 px на спрайтах персонажей и иконках предметов (как в Darkest Dungeon). На фонах обводки не используются.
- Тоновая идея: тёплые «грязные» тона на мире и предметах; яркие/насыщенные тона — только на UI-акцентах, тирах и подсветках.
- НЕ пиксель-арт, НЕ flat cartoon, НЕ chibi, НЕ фэнтези, НЕ аниме.

## Шрифты

Подключаются Engineer'ом через Phaser/Vite. Artist отвечает только за выбор и рекомендации.

| Назначение | Шрифт | Размер (mobile-first) |
|---|---|---|
| Заголовки / акценты | Oswald (или Impact-style fallback) | 24–36 px |
| Основной текст | Roboto (или PT Sans fallback) | 14–18 px |
| Числа (урон, вес, HP) | Roboto Mono (моноширинный) | 14–20 px |

Требования: только без засечек, читаемо на 360×640 dpr 2.0, контрастно к `#1A1A1A` и `#2D2D2A`.

## Размеры спрайтов

| Тип | Размер (px) | Формат | Прозрачный фон |
|---|---|---|---|
| Портрет персонажа (герой, NPC) | 128 × 128 | PNG | Да |
| Спрайт моба | 128 × 128 | PNG | Да |
| Иконка предмета (inventory, crafting) | 64 × 64 | PNG | Да |
| Фон зоны | 800 × 600 | PNG (M2+: WebP при экономии бюджета) | Нет (плотный) |
| UI-кнопка | tap target ≥ 48 × 48 | PNG | Да |

Спрайт-атлас и упаковка — задача M2+. На M1 ассеты лежат одиночными PNG.

## UI-элементы

На M1 UI-кит **не делается** (это M2–M3). Зафиксированы только общие правила, которые в M2+ перейдут в кит:

- Кнопки: тёмная заливка (`#2D2D2A`), песочная подпись (`#D4C5A0`), радиус скругления 4 px, обводка 2 px цвета `#4A4A3A`.
- Активное состояние: подсветка цветом тира (`#C5A267` для основной CTA).
- HP/энергия: тонкие горизонтальные бары с тёмной рамкой, заливка цветами выше.
- Иконки UI наследуют размер иконок предметов (64 × 64), если иное не указано GD.

## Правила стиля

1. Все спрайты — на прозрачном фоне (`alpha = 0` вне силуэта).
2. Контрастные чёрные обводки 2–3 px на спрайтах персонажей и иконках предметов.
3. Тёплые/грязные тона для мира и инвентаря; яркие — только для тиров, состояний и CTA.
4. Никаких градиентов в стиле web-2.0 на UI; допустимы painterly-градиенты внутри силуэта спрайта.
5. Все названия файлов — `snake_case`, ASCII, без пробелов: `wood.png`, `forest.png`, `gunpowder.png`.
6. Расширения: PNG для всего на M1. WebP допускается на M2+ при превышении бюджета.

## Референсы

- **Darkest Dungeon** — атмосфера, портреты, контраст света/тени.
- **This War of Mine** — тема, настроение, тёплые «грязные» тона.
- **Day R Survival** — inventory UI, плотность ячеек, иконки ресурсов.
- **Escape from Tarkov** — инвентарь, модули оружия (вдохновение, не копия).

## Арт-пайплайн

Пайплайн фиксируется здесь как **процесс для M2+**. На M1 ассеты собраны как минимальные placeholder'ы в стилистике этого style-guide (см. секцию «M1 placeholder pipeline» ниже), AI-генерация ещё не подключена.

### Пайплайн для M2+ (AI-генерация)

1. **Генерация** через Stable Diffusion / DALL-E / Midjourney.
2. **Промпт-шаблоны:**
   - Персонаж: `military graphic novel illustration, full body tactical operator, dark olive uniform, bold black outline 2-3px, painted style, Darkest Dungeon atmosphere, contrasted shadows, transparent background, 128x128 sprite`
   - Иконка предмета: `military graphic novel icon, [item name], single object centered, bold black outline 2-3px, dirty palette olive rust dark green, painted style, transparent background, 64x64`
   - Фон зоны: `military graphic novel matte painting, dark post-apocalyptic [biome], heavy shadows, dirty green and rust palette, no characters, no UI, 800x600, landscape`
3. **Пост-обработка:**
   - Удаление фона (rembg / Photoshop / GIMP), alpha-cleanup.
   - Нормализация палитры под список выше (color match / selective hue в Affinity/PS).
   - Масштабирование до целевого размера, без bilinear-блюра по контуру.
   - Контурная докраска bold-обводки 2–3 px на спрайтах персонажей и иконках.
4. **Экспорт:**
   - PNG для спрайтов и иконок (`PIL.Image.save(..., optimize=True)`).
   - WebP для фонов на M2+, если PNG > бюджет (`PIL.Image.save(..., quality=80, method=6)`).
5. **Нейминг:** `{type}_{name}.png`, snake_case, ASCII. Альтернативно — короткий `{name}.png` в соответствующей подпапке (`assets/sprites/items/wood.png`).

### M1 placeholder pipeline (что использовано в этой вехе)

- **Инструмент:** Pillow (Python), программная отрисовка по координатам.
- **Решение PM (2026-05-19):** для M1 разрешена Pillow-генерация placeholder'ов, чтобы разблокировать Engineer-интеграцию (`staff/decisions/DECISIONS.md`).
- **Соответствие style-guide:** палитра, размеры, прозрачный фон, контрастные обводки соблюдены.
- **Что не делается на M1:** AI-генерация, UI-кит, спрайты мобов, анимации, атласы. Это M2+.
