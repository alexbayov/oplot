# Style Guide — «Оплот»

> Визуальные стандарты проекта. Artist заполняет на M1.

## Визуальный стиль

**"Military Graphic Novel"**
- Darkest Dungeon (атмосфера, контрастные тени) + Modern Military (сеттинг)
- Bold линии (2-3px), painted style
- НЕ пиксель-арт, НЕ flat cartoon, НЕ фэнтези
