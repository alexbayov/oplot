# Game Design Document — «Оплот»

> Этот документ — **единственный источник правды** по игровым механикам.
> Если чего-то нет здесь — этого нет в игре.
> Все числа — в `balance.md`.

## High Concept

Одиночка выживает в заброшенном городе после техногенной катастрофы. Core loop: вылазки за ресурсами → пошаговый бой → лут с системой веса → крафт модульного оружия и брони → развитие базы (Оплот).

## Платформа и ограничения

- Яндекс.Игры (HTML5)
- Mobile-first (портрет 375x667+)
- Bundle < 5 MB
- Phaser 3 + TypeScript + Vite
