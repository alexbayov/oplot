# Бэклог (пост-релиз)

> Фичи, которые **не входят** в M0–M9. Будут реализованы после релиза на Яндекс.Играх.

## Приоритет 1 (первые обновления)

- [ ] **Коммуна / NPC-члены** — набор людей в оплот, бонусы от членов, управление
- [ ] **Сюжетная линия** — квестовая цепочка с финалом
- [ ] **Новые зоны** — расширение карты за пределы города

## Приоритет 2

- [ ] **PvP-рейды** — нападение на чужие оплоты (если мультиплеер)
- [ ] **Сезонные ивенты** — временные зоны, уникальный лут
- [ ] **Лидерборды** — рейтинг выживших

## Приоритет 3

- [ ] **Транспорт** — велосипед, мотоцикл (увеличивает грузоподъёмность на вылазке)
- [ ] **Торговля** — NPC-торговцы с уникальным ассортиментом
- [ ] **Питомцы** — собака-компаньон (таскает лут, предупреждает о засадах)

## Отклонённые идеи

| Идея | Причина отклонения |
|---|---|
| Магия / фэнтези | Противоречит сеттингу |
| 3D / изометрия | Не влезет в 5 MB |
| Реальные ТМ оружия | Юридические риски |
| Мультиплеер (MVP) | Scope creep |
