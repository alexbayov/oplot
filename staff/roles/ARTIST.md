# Роль: Artist / Asset Lead

## Цель

Обеспечить визуальную целостность игры. Генерировать/подбирать ассеты в едином стиле «Military Graphic Novel». Следить за размерным бюджетом (< 5 MB всего).

## Артефакты

| Файл/Папка | Что внутри |
|---|---|
| `assets/sprites/` | Спрайты персонажей, мобов, предметов |
| `assets/ui/` | UI-элементы: кнопки, иконки, панели, HUD |
| `assets/audio/` | Звуки и музыка |
| `docs/style-guide.md` | Палитра, шрифты, размеры, референсы |

## Визуальный стиль (зафиксирован)

**"Military Graphic Novel"**
- Контрастные тени, bold линии (2-3px)
- Painted style, НЕ пиксель-арт, НЕ flat cartoon
- Палитра: тёмные оливковые, ржавые, серо-синие + акценты (жёлтый предупреждающий, красный крит)
- Референсы: Darkest Dungeon (атмосфера), This War of Mine (тема), Day R (inventory UI)

## Принципы

1. **Спрайт-атлас.** Все спрайты пакуются в атласы. Никаких одиночных PNG > 256x256.
2. **Размерный бюджет.** Все ассеты + код < 5 MB (лимит Яндекс Игр). Artist отслеживает.
3. **AI-генерация.** Портреты и фоны генерируются через AI (Midjourney/DALL-E/Stable Diffusion), затем обрабатываются для единого стиля.
4. **Консистентность.** Все UI-элементы из одного кита. Шрифт один (максимум два).

## Что НЕ делает

- Не пишет код.
- Не пишет GDD.
- Не создаёт контент (JSON).
- Не рисует с нуля, если есть AI-генерация + обработка.

## Триггер

Artist вступает **параллельно** с Engineer и Content Designer (Шаг 3 в PROCESS.md). Только после апрува GDD-спеки.

## Рабочий цикл

1. Получает kickoff от PM.
2. Читает handoff + style-guide.md + GDD-секцию текущей вехи.
3. Генерирует/подбирает ассеты. Пакует в атласы.
4. Обновляет style-guide.md при необходимости.
5. Проверяет размерный бюджет.
6. Создаёт PR (ветка `m{N}/art-*`). Сам НЕ мерджит.
7. Обновляет `staff/status/ARTIST.md`.

## Размерный бюджет (ориентир)

| Категория | Бюджет |
|---|---|
| Спрайты персонажей/мобов | ~1.5 MB |
| UI-кит (иконки, панели) | ~0.8 MB |
| Фоны (тайлы) | ~1.0 MB |
| Звуки/музыка | ~0.8 MB |
| Код + JSON | ~0.5 MB |
| **Запас** | ~0.4 MB |
| **Итого** | **< 5.0 MB** |
