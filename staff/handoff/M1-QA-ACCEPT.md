## Твоя задача

Проверить 3 PR по чек-листам ниже. Для каждого PR: APPROVE или CHANGES_REQUESTED.

---

### Чек-лист 1: Engineer PR (`m1/eng-bootstrap`)

#### Билд
- [ ] `npm install` — без ошибок
- [ ] `npm run dev` — запускается, браузер показывает экран
- [ ] `npm run build` — билд проходит
- [ ] `npm run lint` — без ошибок (или минимальные warnings)
- [ ] `npm run typecheck` — без ошибок

#### Сцены
- [ ] BootScene существует и загружается первой
- [ ] BaseScene: видно текст "Оплот" или аналогичный, есть кнопки навигации
- [ ] MapScene: есть кнопка/ссылка на зону
- [ ] SortieScene: заглушка присутствует
- [ ] CombatScene: заглушка присутствует, есть кнопки действий
- [ ] InventoryScene: заглушка присутствует
- [ ] CraftScene: заглушка присутствует

#### Типы
- [ ] `src/types/` содержит интерфейсы Item, Mob, Recipe, Zone
- [ ] Интерфейсы соответствуют JSON-схемам из GDD.md

#### Код
- [ ] Нет `any` типов
- [ ] Нет закомментированного кода
- [ ] .gitignore содержит node_modules/ и dist/
- [ ] `dist/` размер < 1 МБ (после `npm run build`)

---

### Чек-лист 2: Content PR (`m1/content-mvp`)

#### items.json
- [ ] 15 предметов (8 ресурсов + 2 оружия + 2 брони + 3 расходника)
- [ ] Все поля заполнены по JSON-схеме из GDD
- [ ] Каждый предмет механически уникален (не просто другое имя)
- [ ] Вес соответствует balance.md

#### mobs.json
- [ ] 3 моба
- [ ] Каждый механически уникален (разное поведение, не только числа)
- [ ] Статы соответствуют balance.md

#### recipes.json
- [ ] 5 рецептов
- [ ] Ингредиенты ссылаются на id из items.json
- [ ] Результаты ссылаются на id из items.json

#### zones.json
- [ ] 1 зона (Лес)
- [ ] 3 уровня глубины
- [ ] Мобы ссылаются на id из mobs.json
- [ ] Ресурсы ссылаются на id из items.json

#### Anti-scope
- [ ] НЕТ радио-сигналов
- [ ] НЕТ перков
- [ ] НЕТ больше 1 зоны

---

### Чек-лист 3: Artist PR (`m1/art-initial`)

#### style-guide.md
- [ ] Описан общий стиль (Military Graphic Novel)
- [ ] HEX-коды палитры указаны
- [ ] Размеры спрайтов указаны
- [ ] Шрифты указаны
- [ ] Арт-пайплайн описан

#### Ассеты
- [ ] `hero.png`: 128x128, прозрачный фон
- [ ] 8 иконок ресурсов: 64x64, различимы
- [ ] `forest.png`: ~800x600, тёмный стиль
- [ ] Стиль соответствует style-guide
- [ ] НЕ пиксель-арт
- [ ] Общий размер < 300 КБ
