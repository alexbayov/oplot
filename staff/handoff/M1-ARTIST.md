## Твои deliverables

### 1. `docs/style-guide.md` — Visual Style Guide

Заполни документ с описанием визуального стиля:

#### Общий стиль
- "Military Graphic Novel"
- Референсы: Darkest Dungeon (контрастные тени, bold линии) + современный военный сеттинг
- НЕ пиксель-арт, НЕ чиби, НЕ мультяш
- Полуреалистичный рисунок с претензией на реал

#### Палитра (обязательно HEX-коды)
- Фон: `#1A1A1A` (тёмный), `#2D2D2A` (тёмно-серый)
- Основные: `#4A4A3A` (хаки), `#8B7355` (ржавчина), `#3D5C3A` (тёмно-зелёный)
- Тиры предметов: серый T1, `#4CAF50` T2, `#2196F3` T3, `#9C27B0` T4, `#FF9800` T5
- UI текст: `#D4C5A0` (песочный)
- HP бар: `#8B0000`
- Энергия: `#4682B4`

#### Размеры спрайтов
- Персонаж (герой, мобы): 128x128 px
- Иконки предметов: 64x64 px
- Фоны: 800x600 px (scaled по нужде)
- UI кнопки: по контексту, минимум 48x48 tap target

#### Шрифты
- Основной: без засечек, читаемый на мобилке (рекомендация: Roboto или PT Sans)
- Акцентный: для заголовков (рекомендация: Oswald или Impact-стиль)

#### Правила стиля
- Все спрайты на прозрачном фоне
- Контрастные чёрные обводки (как в Darkest Dungeon)
- Тёплые/грязные тона для мира
- Яркие тона только для UI-акцентов и подсветки тиров

### 2. Placeholder-ассеты

На M1 нужны **минимальные ассеты** чтобы Engineer мог подключить их в Phaser:

#### Спрайт героя (`assets/sprites/hero.png`)
- 128x128, PNG, прозрачный фон
- Человек в тактической одежде, вид сбоку/3/4
- Стиль: illustrated, тёмные тона, чёткие контуры

#### 8 иконок ресурсов (`assets/sprites/items/`)
Каждая 64x64, PNG, прозрачный фон:
- `wood.png` — деревяшки/доски
- `scrap.png` — куски металла
- `cloth.png` — ткань/тряпки
- `food.png` — консервная банка
- `water.png` — бутылка воды
- `gunpowder.png` — мешочек пороха
- `leather.png` — кусок кожи
- `rope.png` — моток верёвки

#### Фон Леса (`assets/backgrounds/forest.png`)
- 800x600, PNG или WebP
- Густой тёмный лес, постапокалиптический стиль
- Тёмная палитра с зелёными/коричневыми тонами

### 3. Арт-пайплайн (описание в style-guide.md)

Опиши процесс создания ассетов:
1. AI-генерация через Stable Diffusion / DALL-E / Midjourney
2. Промпт-шаблоны для каждого типа (персонаж, иконка, фон)
3. Пост-обработка: удаление фона, нормализация палитры, масштаб
4. Экспорт: PNG для спрайтов, WebP для фонов
5. Именование: `{type}_{name}.png` (snake_case)
