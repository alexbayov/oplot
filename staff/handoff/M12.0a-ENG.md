## TL;DR

Skeleton M12 Combat Engine — фасад поверх старой `combat.ts`. **Структура без поведения**: все будущие подвехи M12.0b-c и M12.1-M12.6 наполняют **поведение**, не структуру.

Старый бой (CombatScene) **не тронут** — engine существует параллельно, его интеграция в M12.0c.

## Что внутри (новые файлы)

1. `src/systems/combatTypes.ts` (228 строк) — все shape'ы:

## API контракт (для M12.0b и далее)

CombatEngine использовать так:

```ts
const state = createState({ zoneId: "forest", scenario: "sortie" });
state.actors = [
  createActor({ id: "hero", kind: "hero", nameRu: "Игрок", maxHp: 100, equipped: ... }),
  createActor({ id: "m1", kind: "mob", nameRu: "Мародёр", maxHp: 20 }),
];
const engine = new CombatEngine(state);

engine.startTurn();
const r = engine.resolveHeroAction({ actorId: "hero", kind: "attack", targetId: "m1" });
if (r.ok) renderLog(r.logEntries);
const mobActions = engine.resolveMobActions({ ... });
const outcome = engine.endTurn();
if (outcome.combatEnded) endCombat(outcome.outcome);
```

## Что НЕ сделано (anti-scope для M12.0a)

- CombatScene **не переключена** на engine (это M12.0c)
- Mob damage roll в `resolveMobActions` — stub (это M12.0b)
- Mod effects не агрегируются в damage calc (это M12.0b)
- Durability не уменьшается при атаке (это M12.0b)
- Targeting UI / HP-бары / weapon tooltip (это M12.0c)
