## Твои deliverables (что должно быть в PR)

### 1. `docs/GDD.md` — Game Design Document MVP

Опиши следующие системы **только в скоупе M1-M2** (MVP):

#### a) Вылазка (Sortie)
- Игрок выбирает зону на карте → переход на экран вылазки
- В MVP: только 1 зона — **Лес** (3 уровня глубины)
- Каждый уровень = серия боёв (1-3 моба) + лут
- После боёв: экран лута (выбор что взять, ограничение по весу)

#### b) Бой (Combat)
- Пошаговый: ход игрока → ход врагов
- 2 типа оружия: **ближний бой** (нож/дубинка) + **стрелковый** (самодельный пистолет)
- 3 действия: Атака (оружием), Укрытие (+50% защиты на 1 ход), Аптечка (тратит расходник)
- Формулы:
  ```
  Урон = base_damage * random(0.85, 1.15)
  Итого = max(1, Урон - armor_value)
  Инициатива = base_speed - (current_weight / max_weight) * 50
  ```
- Победа = все мобы мертвы → экран лута
- Поражение = HP героя = 0 → теряешь 50% лута, возврат на базу

#### c) Мобы (3 шт для MVP)
- **Мародёр:** HP 30, Урон 5-8, XP 10, дроп: Ткань/Еда, редкий: Патроны 15%
- **Дикий пёс:** HP 20, Урон 8-12, XP 8, дроп: Кожа, быстрый (высокая инициатива)
- **Мутант:** HP 60, Урон 10-15, XP 25, дроп: Хим.реагент, редкий: Катализатор 5%

#### d) Инвентарь и вес
- Макс. вес рюкзака: 30 кг (начальный)
- Каждый предмет имеет вес
- Текущий вес влияет на инициативу в бою и скорость возврата
- При перегрузе (>100%): нельзя бежать, инициатива = 0

#### e) Крафт (базовый)
- Мастерская на базе
- 5 рецептов MVP (см. ниже в balance.md)
- Вход: конкретные ресурсы в конкретном количестве
- Выход: конкретный предмет
- Крафт мгновенный в MVP (таймеры — M3+)

#### f) JSON-схемы
Опиши TypeScript-интерфейсы для:
```typescript
interface Item {
  id: string;
  name: string;
  type: "resource" | "weapon_melee" | "weapon_ranged" | "armor" | "consumable" | "blueprint";
  tier: 1 | 2 | 3 | 4 | 5;
  weight: number;
  // ... другие поля по типу
}

interface Mob { ... }
interface Recipe { ... }
interface Zone { ... }
```

### 2. `docs/balance.md` — Черновой баланс MVP

Таблицы с конкретными числами:
- Статы героя (начальные): HP, EN, скорость, макс. вес
- Статы 3 мобов: HP, урон, инициатива, XP, дроп
- Статы оружия MVP: нож, дубинка, самодельный пистолет
- Статы брони MVP: тканевая куртка
- 5 рецептов:
  1. Дубинка = Дерево x3 + Верёвка x1
  2. Заточка = Металлолом x2
  3. Самодельный пистолет = Металлолом x5 + Дерево x2 + Порох x3
  4. Бинт = Ткань x3
  5. Аптечка = Бинт x2 + Медикаменты x1
- Вес каждого ресурса
- XP-таблица: сколько XP на каждый уровень (1-5 для MVP)
