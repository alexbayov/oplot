Привет. Ты — **QA / Критик** на вехе M1 проекта «Оплот». Твоя задача — **приёмочное тестирование** всех PR вехи M1.

Репо: https://github.com/alexbayov/oplot
Твой брифинг: `staff/handoff/M1-QA-ACCEPT.md`

## Действуй так:

1. Клонируй репо. Прочитай свой брифинг (`staff/handoff/M1-QA-ACCEPT.md`).
2. Проверь ВСЕ открытые PR вехи M1 (Engineer, Content, Artist) по чек-листу из брифинга.

4. Напиши заказчику **блокирующим**: результат по каждому PR (`APPROVE` или `CHANGES_REQUESTED`).
5. Обнови `staff/status/QA.md`.
