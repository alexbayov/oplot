### 2. baseCritChance в каждом стволе
Поле `base_crit_chance` для всех 30 weapons (22 drop + 8 craft).
Источник — `docs/balance.md` секция M12.5.

### 3. Финальные mod effects
8 модов в items.json. Текущие placeholder'ы → значения от GD.
Поле `effects: { damage_delta, noise_set, accuracy_delta, mag_delta, crit_bonus }`

### 4. Durability финальные значения
`durability_max` для всех 30 оружий.

### 5. activeAbility ref для каждого ствола
**Только** ссылка на ability ID. Сам список abilities — М12.1. Поле `active_ability_id: "ak_burst"` и т.п.
