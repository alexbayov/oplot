---

## Anti-scope M1

НЕ делаем в M1:
- Никакой игровой логики (бой, лут, крафт) — только заглушки экранов
- Никаких реальных спрайтов мобов (только герой + ресурсы + фон)
- Никакого Yandex SDK
- Никаких анимаций
- Никакого радио
- Никаких перков
- Никаких звуков

---

## Definition of Done (M1)

1. `npm run dev` запускается, в браузере видно экран "Оплот" (BootScene → MapScene)
2. `npm run build` проходит без ошибок
3. `npm run lint` и `npm run typecheck` чисто
4. GDD.md содержит описание MVP механик
5. balance.md содержит черновые цифры
6. content/*.json заполнены MVP-данными
7. style-guide.md описан
8. Хотя бы 1 спрайт + 8 иконок + 1 фон в assets/
9. QA approve на все PR
