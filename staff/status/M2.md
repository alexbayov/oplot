## Скоуп M2 (из PLAN §2)

Играбельный MVP. Игрок может:

1. Открыть игру (загрузка < 5 сек на мобильном 4G).
2. Увидеть базу (экран Оплота) с кнопкой «Вылазка».
3. Выбрать зону «Лес» (единственная).
4. Встретить 1–3 мобов, провести пошаговый бой (нож + самодельный пистолет).
5. Собрать лут (3–5 ресурсов), видеть вес в рюкзаке.
6. Вернуться на базу, увидеть инвентарь.
7. Скрафтить 1–2 предмета из собранных ресурсов.

## Anti-scope M2

- Нет новых зон, мобов, предметов, рецептов — используем только canonical content из M1 (`content/*.json`).
- Нет радио (M6).
- Нет перков (M4).
- Нет боссов (M5).
- Нет модулей оружия (M3+).
- Нет коммуны (post-release).
- Нет Yandex SDK (M8).
- Нет реальных анимаций / звуков (M7).
- Нет нового арта (placeholder M1 ассеты остаются; Artist на M2 НЕ запускается без явной необходимости).
