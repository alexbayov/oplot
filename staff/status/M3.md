## Скоуп M3 (из PLAN §3)

«Расширение мира». Игрок может:

1. Вылетать в 3 разные зоны: **Лес** (есть), **Склад** (новая), **Город** (новая) — каждая со своим биомом, мобами и ресурсами.
2. Встречать **8 типов мобов** (3 есть: marauder/wild_dog/mutant; +5 новых: см. M3 GD-amendment).
3. Крафтить из **15 рецептов** (5 есть; +10 новых: оружие T2, броня T2, расходники, инструменты).
4. Открывать **RadioScene** — UI-заглушка: вкладка «Радио» на базе, список входящих сигналов (пока пустой / 1-2 dummy-сигнала), кнопки «Принять» / «Игнорировать» — обе **только меняют local state без последствий** (полная логика — M6).

## Anti-scope M3

- Нет перков / уровней / XP-системы (M4).
- Нет боссов / инстансов / T3 чертежей (M5).
- Нет полной радио-логики: сигналы с ветвлениями, засады, награды, фракционные репутации, доверие (M6).
- Нет модулей оружия (отложено: либо M3 stretch, либо M5).
- Нет коммуны / NPC-членов (post-release).
- Нет Yandex SDK (M8).
- Нет реальных анимаций / звуков / эффектов (M7).
- Нет новых стилей UI или сторонних UI-библиотек (только Phaser built-in).

## Что уже готово (унаследовано с M2)

- Core loop полный: Base → Map → Sortie → Combat → Loot → Return → Base + Inventory + Craft.
- 4 системы: combat, weight, loot, craft.
- 9 сцен зарегистрированы в `src/main.ts`.
- 15 items / 3 mobs / 5 recipes / 1 forest zone в `content/*.json`.
- `src/types/{item,mob,recipe,zone}.ts` соответствуют GDD §6.
- `src/state/GameState.ts` — runtime state со всеми нужными полями.
- 49 vitest unit-тестов на формулы (combat / weight / craft / loot / return_time).
- 10 placeholder-ассетов (hero + 8 item icons + forest bg) + Pillow-пайплайн в `assets/`.

## Definition of Done для M3 (из PLAN §5 + §3)

1. GDD расширен под M3: §5 (5 новых мобов), §6.4 (2 новые зоны), новая §7 (radio system structure).
2. `balance.md` имеет числа для всех 5 новых мобов и 10 новых рецептов.
3. `content/*.json` содержит **минимум**: items ≥ 30 (15 → ~30), mobs = 8, recipes ≥ 15, zones = 3, radio signals ≥ 2 dummy.
4. Engineer:
   - MapScene показывает 3 зоны, переключение между ними работает.
   - 5 новых mob AI behaviors реализованы и покрыты vitest тестами.
   - RadioScene существует как UI-заглушка (никакой реальной логики ветвлений — только show signals + accept/ignore локально).
   - Все 49 M2 тестов плюс новые тесты на M3 mob AI — все зелёные.
5. Artist:
   - 5 mob sprites 128×128, прозрачный фон, стиль соответствует style-guide.
   - 2 background sprites 800×600 (warehouse + city).
   - ~10 новых item icons 64×64 (под новые resources / weapons / armor).
   - Общий ассет-budget ≤ 500 КБ дополнительно (M2 был 81 КБ; M3 цель ≤ 600 КБ суммарно).
6. QA Acceptance прошла smoke + чек-листы M3 (полные §1–§7).
7. Не сломаны M1/M2 артефакты (49 unit-тестов всё ещё проходят, 7-шаговый MVP flow в Forest всё ещё работает).
8. `npm run build` clean, ≤ 2 MB bundle, без console.error.
