«Перки и прогрессия». Игрок может:

1. Получать **XP** за убийство мобов (`mob.xp_reward`).
2. Накапливать XP → уровни по XP-curve `round(40 * level^1.5)`, MAX_LEVEL=10.
3. На каждом level-up получать **выбор из 3 случайных** перков из пула 8 пассивных.
4. Просматривать список взятых перков + текущий уровень + прогресс до следующего на `ProgressionScene`.
5. Видеть применённые модификаторы перков в бою / крафте / возврате.

**8 пассивных перков (финальные числа из balance.md §M4):**

1. `tough_skin` — +15 hp_max (additive)
2. `sharp_blade` — ×1.15 damage (multiplicative)
3. `lean_pack` — ×0.85 weight_penalty_multiplier (multiplicative)
4. `lucky_scavenger` — ×1.20 loot_quantity_multiplier (multiplicative)
5. `keen_eye` — +0.05 crit_chance (additive)
6. `reinforced_plates` — ×1.15 armor_efficiency (multiplicative)
7. `quick_hands` — ×0.90 crafting_speed_multiplier (multiplicative)
