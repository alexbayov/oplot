# Оплот — Post-apoc Shelter Sim

> **Жанр:** Post-apoc shelter sim с глубоким крафтом и вылазками
> **Платформа:** Яндекс.Игры (HTML5, mobile-first, landscape 1280×720)
> **Стек:** Phaser 3 + TypeScript + Vite + Yandex Games SDK

## High Concept

Одиночка строит убежище после техногенной катастрофы. Грядки, генераторы, верстак, койка — база живёт и потребляет ресурсы. Вылазки в зоны тащат то, что базе нужно: воду, топливо, металл, еду, компоненты для крафта. Возвращение, разбор лута, очередь крафта, новый цикл.

Каждая вылазка — выбор цели (тихо / жадно / точечно за конкретным ресурсом), цепочка авторесолв-энкаунтеров с правом остановки на любом этапе, нарративные строки вместо тактического боя. Бой как тактика — отдельный трек на M14+, сейчас намеренно вырезан до авторесолва, чтобы база и крафт стали центром игры.

## Сравнения

- Sheltered
- Fallout Shelter
- This War of Mine (по жанру shelter sim, не по тону)
- State of Decay (shelter loop)

## Опорные слои

- **База** — постройки, ресурсы, офлайн-прогрессия (M13 PR-5).
- **Крафт** — компоненты + слоты + аффиксы в стиле Lineage 2 (M13 PR-3/PR-4).
- **Вылазки** — авторесолв с decision surface (цель, расходники, «идти/вернуться»). Сделано в M13 PR-1.
- **Карта** — три открытые зоны на лаунч с разными loot-профилями (M13 PR-2).

См. [`docs/redesign/M13-PIVOT.md`](docs/redesign/M13-PIVOT.md) для полного плана редизайна.
